Visual Tool - How to use

First load the video, using the Choose File button (below the video frame) and click the button Load Video (three buttons left) Start playing the video

The keyboard keys from 1 to 6 represent, respectively each Team Member from A to F. When you press the key, the timer starts and when you release the key, the timer finishes. Then the timestamps will appear on a table on the right side.

If any mistake is made, you can stop the video, delete the timestamp on the right and rewind it using the Rewind 10s button.

If you want to slow down the video speed, click on the Slow Video button, it will make the video speed to 0.75. Clicking once again will make the video to normal speed.

After you have finished, write the excel file name in the box above the Export .XLS button and click the Export .XLS button to export the file with the timestamps

Before loading a new video click on Clean Data
